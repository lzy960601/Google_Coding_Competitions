## B

- 显然每一层的池子是共享容积的

## C

- 考虑每个种子都有一个种植ddl，在这个ddl之后种植毫无意义
- 我们把所有种子按照ddl从后向前排序，枚举当前ddl下的种子该如何种植
- 显然种植价值最大的那些种子
- 所以用一个堆维护当前可以种的种子，贪心种植即可
